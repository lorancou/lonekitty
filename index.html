Lone Kitty

Kitty feels lonely. Help him find some joy with your keyboard's arrow keys.

A Lumdum Dare #22 game entry. This is Javascript/HTML5/canvas stuff, so this should work with a recent version of any modern browser (Firefox, Chrome...). You have a slight chance to play with IE too, thanks to excanvas. Sources on github.
